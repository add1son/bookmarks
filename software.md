# My favorite Software & Web Extensions

## Platform Agnostic (Cross-Platform)

### Productivity & Notes
* [Obsidian](https://obsidian.md/) - Knowledge base mapper and markdown notebook.
* [Zim](https://zim-wiki.org/) - A graphical text editor used to maintain a collection of wiki pages.
* [RedNotebook](https://rednotebook.app/downloads.html) - Desktop diary and journal software with tagging support.

### Media, Audio & Transcription
* [Calibre](https://calibre-ebook.com/) - Comprehensive open-source e-book manager and viewer.
* [Mumble](https://www.mumble.info/) - Low-latency, high-quality open-source voice chat utility.
* [Open Broadcaster Software (OBS)](https://obsproject.com/) - Free and open-source software for video recording and live streaming.
* [Avidemux](https://avidemux.sourceforge.net/) - Free video editor designed for simple cutting, filtering, and encoding tasks.
* [LosslessCut](https://github.com/mifi/lossless-cut) - Swiss army knife of lossless video/audio editing based on FFmpeg.
* [yt-dlp](https://github.com/yt-dlp/yt-dlp/) - Feature-rich command-line YouTube/media downloader.
* [beets](https://beets.io/) - Media library management system and metadata tagger for music geeks.
* [Vibe Transcribe](https://thewh1teagle.github.io/vibe/) - Local AI audio transcription engine; efficient desktop alternative to [wisprflow](https://wisprflow.ai/).

### System, Networking & Databases
* [Floorp](https://floorp.app/en/) - Customisable, privacy-focused web browser based on Firefox.
* [KeePass](https://keepass.info/) - Classic, secure local-first password manager.
* [AnyDesk](https://anydesk.com/en) - Fast proprietary remote server/desktop access client.
    * [RustDesk](https://rustdesk.com/) - Excellent FOSS alternative to AnyDesk.
* [Quassel IRC](https://quassel-irc.org/) - Modern, cross-platform, distributed IRC client.
* [Thunderbird](https://www.thunderbird.net/en-US/) - Robust email, calendar, and news feed client.
* [Telegram](https://telegram.org/) - Cloud-based instant messaging and multimedia platform.
* [VirtualBox](https://www.virtualbox.org/) - Multi-platform x86 and AMD64/Intel64 virtualization software.
* [RPCS3](https://rpcs3.net/quickstart) - Open-source Sony PlayStation 3 emulator.
* [balenaEtcher](https://etcher.balena.io) - Streamlined tool to flash OS images to SD cards & USB drives.
* [Portainer](https://www.portainer.io/) - Container management interface for Docker and Kubernetes environments.
* [pgAdmin](https://www.pgadmin.org/) - Feature-rich open administration platform handling PostgreSQL server databases.

### AI 
* [odysseus](https://github.com/pewdiepie-archdaemon/odysseus/) - A self-hosted AI workspace for chat, agents, research, documents, email, notes, calendar, and local model workflows.

## Android (F-Droid Focus)

* [scrcpy](https://github.com/Genymobile/scrcpy) - Display and control Android devices over USB or TCP/IP from a PC.
* [Syncthing](http://f-droid.org/packages/com.nutomic.syncthingandroid/) - Open-source file synchronization engine across local nodes.
* [Primitive ftpd](http://f-droid.org/en/packages/org.primftpd/) - Simple FTP/SFTP server implementation to migrate data off mobile local storage.
* [DNS66](https://f-droid.org/packages/org.jak_linux.dns66/) - DNS-based host adblocking without requiring root privileges.

* [KeePassDX](https://f-droid.org/packages/com.kunzisoft.keepass.libre/) - Multi-format KeePass password manager interface natively built for mobile.
* [Bromite](https://www.bromite.org/fdroid) - Chromium fork featuring built-in adblocking and enhanced privacy engines.
* [Unstoppable Crypto Wallet](https://f-droid.org/en/packages/io.horizontalsystems.bankwallet/) - Decentralized non-custodial crypto-management asset engine.

### Media & News Aggregators

* [Feeder](https://f-droid.org/packages/com.nononsenseapps.feeder/) - Minimalist and fluid local open-source RSS reader.
* [Raven](https://f-droid.org/en/packages/kshib.raven/) - Custom web scraping and API-centric dynamic news reader.
* [Hacki](https://f-droid.org/packages/com.jiaqifeng.hacki/) - Clean, feature-complete client tracking YCombinator Hacker News.
* [Podverse](https://f-droid.org/en/packages/com.podverse.fdroid/) - Open-source podcast aggregator with clip sharing features.
* [MPV Android](http://github.com/mpv-android/mpv-android/releases) - Video player port built around libmpv.
* [Simple Music Player](https://f-droid.org/en/packages/com.simplemobiletools.musicplayer/) - Clean offline audio file playback engine.
* [Vanilla Music](https://f-droid.org/packages/ch.blinkenlights.android.vanilla/) - Clean, extensible local audio manager.

### Utilities & Tools
* [Element](http://f-droid.org/packages/im.vector.app/) - Secure matrix messaging wrapper.
* [K-9 Mail](https://f-droid.org/packages/com.fsck.k9/) - Production-ready open email engine (Now the foundation of Thunderbird Mobile).
* [Open Note Scanner](https://f-droid.org/packages/com.todobom.opennotescanner/) - Document imaging tool converting handwritten pages to image assets.

* [SherpaTTS](https://f-droid.org/en/packages/org.woheller69.ttsengine/) - Offline local text-to-speech architecture leveraging Piper & Coqui engines.
* [Whisper+](https://f-droid.org/en/packages/org.woheller69.whisperplus/) - Voice-to-text input method keyboard layout powered locally by OpenAI Whisper models.
* [Sidestep](https://apt.izzysoft.de/fdroid/index/apk/com.blankdev.sidestep) - Intercepts URLs and redirects them to alternative frontends & removes tracking

## Windows

### Terminal, SSH & Utilities
* [mRemoteNG](https://mremoteng.org/) - Tabbed, multi-protocol remote connections manager.
* [WindTerm](https://github.com/kingToolbox/WindTerm) - High-performance, advanced SSH/Telnet/Serial terminal emulator. Interfaces with KeePassDX for SSH key-based auth.
* [RDCMan (Remote Desktop Connection Manager)](https://docs.microsoft.com/en-us/sysinternals/downloads/rdcman) - Lightweight Sysinternals interface managing grouped RDP assets.
* [Notepad++](https://notepad-plus-plus.org/) - Highly extensible source code editor and Notepad replacement.
* [Fastfetch](https://github.com/fastfetch-cli/fastfetch) - Like neofetch, but modern and lightning-fast. Install via `winget install fastfetch`.

### System & Automation
* [AutoHotkey](https://www.autohotkey.com/) - Powerful macro and hotkey automation script processor.
* [Ditto Clipboard Manager](https://ditto-cp.sourceforge.io/) - Database extension back-end syncing and storing system clipboard assets.
* [PowerToys](https://learn.microsoft.com/en-us/windows/powertoys/install) - Microsoft system extension toolkit containing FancyZones, PowerRename, and OCR tools.
* [Snappy Driver Installer](https://www.snappy-driver-installer.org/) - Accurate open driver indexing wizard engine for offline clean deployment.
* [Hiren's BootCD PE](https://www.hirensbootcd.org/) - Complete WinPE recovery emergency platform for bootable triage.
* [Hyper-V](https://learn.microsoft.com/en-us/virtualization/hyper-v-on-windows/) - Enterprise-grade native hypervisor virtualization layer built right into Windows.

### File & Archive Management
* [7-Zip](http://7-zip.org) - Open file archiver with extremely high compression efficiency ratios.
* [Everything by Voidtools](https://www.voidtools.com/) - Near-instant real-time administrative system file locator indexed by NTFS MFT data.
* [FileConverter](https://github.com/Tichau/FileConverter) - Shell extension allowing deep system context menu right-click file conversions.
* [Files App](https://files.community/docs/install) - Fluent design replacement for Explorer.exe featuring system tabs and tags.
* [Directory Opus](https://www.gpsoft.com.au/) - Ultimate file-system environment manager optimized explicitly for power-users.
* [WinMerge](https://winmerge.org/?lang=en) - Visual differencing and merging utility for files and directories.

### Graphics & Imaging
* [ImageGlass](https://imageglass.org/) - Clean, lightweight open-source image viewer built for modern workflows.
* [DigiKam](https://www.digikam.org/) - Professional open-source asset management suite processing photos and RAW files.
* [XnView MP](https://www.xnview.com/en/xnviewmp/) - Versatile and powerful media browser, viewer, and converter.
* [IrfanView](https://www.irfanview.com/) - Ultra-fast, legacy graphics viewer supporting massive media file formats and batch processing.
* [Paint.NET](https://www.getpaint.net/) - Highly performant, free layer-based raster graphics editor stepping past basic MSPaint formats.
* [ShareX](https://getsharex.com/) - Definitive capture utility allowing advanced custom workflow automation capturing video or screenshots. *(Greenshot alternative)*

### Audio, Video & AI Tools
* [VLC Media Player](http://videolan.org/vlc/) - Open-source modular engine playing almost any multimedia container file codec native.
* [Foobar2000](https://www.foobar2000.org/) - Advanced component modular audio manager designed for metadata adjustments.
* [Exact Audio Copy (EAC)](https://www.exactaudiocopy.de/) - Renowned secure audio CD digital extraction grabber.
* [yt-dlp-interface](https://github.com/ErrorFlynn/ytdlp-interface) - Fluent, approachable GUI shell wrapper backing yt-dlp backend deployments.
* [Ultimate Vocal Remover (UVR)](https://ultimatevocalremover.com/) - Advanced neural network tool isolating instruments and vocal tracks from core mixes.
* [Pinokio](https://pinokio.computer/) - One-click program orchestrating automated configuration setups of complex AI applications and local LLMs locally.
* [Notion](https://www.notion.so/product) - Modular workspace planning layout canvas.
    * [Zettlr](https://www.zettlr.com/download/win32) - Academic Markdown editor deployed locally as a privacy-focused alternative.
